# Skill: Docusaurus & Robotics Theme Design

## 1. Aesthetic Goal
* **Theme**: "Physical AI & Cybernetics".
* **Colors**: Use Slate Gray backgrounds, Electric Blue accents (#007AFF), and high-contrast text.
* **Vibe**: Academic yet futuristic (like OpenAI or NVIDIA documentation).

## 2. Components
* **Hero Section**: Must have a strong tagline, a "Get Started" button, and a robotic illustration/gradient.
* **Navbar**: Clean, sticky, with links to "Chapters", "Labs", and "GitHub".
* **Admonitions**: Customize :::tip, :::warning to look modern (rounded corners, soft icons).

## 3. Technical Rules
* Modify `src/css/custom.css` for global styles.
* Modify `docusaurus.config.ts` for layout/navbar/footer.
* Use Tailwind classes if configured, otherwise standard CSS variables.
