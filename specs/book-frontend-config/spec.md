## Feature Specification: Book Frontend Configuration Update

### User Story

As a course administrator, I want to update the Docusaurus site's title and tagline so that the book accurately reflects its content and purpose.

### Requirements

- **P1: Update Site Title**: The Docusaurus site title in `book_source/docusaurus.config.ts` MUST be set to "Teaching Physical AI & Humanoid Robotics".
- **P1: Update Site Tagline**: The Docusaurus site tagline in `book_source/docusaurus.config.ts` MUST be set to "From Isaac Sim to Real World".
